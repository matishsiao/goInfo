# Introduction:
  GoInfo is get os platform information coding by Golang.
  
  It can help you to know os information.
  
  
## Version:

version:0.0.1

## Futures

get linux information

get windows information

get osx information

get freebsd information

get netbsd information

get openbsd information
  
  
## Install:
```sh
  go get github.com/matishsiao/goInfo
  go build
```

## Struct:
```sh
  type GoInfoObject struct {
	GoOS string
	Kernel string
	Core string
	Platform string
	OS string
	Hostname string
	CPUs int
}
```

## Example:

```sh   
   package main

   import (
	   "github.com/matishsiao/goInfo"
   )

   func main() {
		gi := goInfo.GetInfo()
		gi.VarDump()
	 }
}
```

It's will show:

```sh
   GoOS: linux
   Kernel: Linux
   Core: 3.13.0-27-generic
   Platform: x86_64
   OS: GNU/Linux
   Hostname: ubuntu
   CPUs: 1
```
